# SCP-2719
                        


**项目编号：** SCP-2719

**项目等级：** Keter

**特殊收容措施：** SCP-2719应被保留在内部。

**描述：** SCP-2719为单个抽象概念类构造函数指针变量。受SCP-2719作用的概念会进入或成为内部。同时具有智能与生物性质的人员不得获取有关SCP-2719的详细资料。
