# SCP-CN-769

**项目编号：** SCP-CN-769


**项目等级：** Safe

**特殊收容措施：** SCP-CN-769当前被放置在[Site-CN-03](//scp-wiki-cn.wikidot.com/site-cn-03)低威胁类异常项目收容区（B-2区）中等威胁类异常项目收容区（B-3区）的一个标准异常物品收容箱内。对项目的实验需要至少2名3级或以上人员批准。
20██-██-██更新：当前已无限期停止对项目的一切实验。

**描述：** SCP-CN-769是一个高23厘米，最大外径96厘米，重1.7千克的木盆。项目的异常性质在于，当一名使用者向SCP-CN-769内部投入<sup class='footnoteref'>
 <a shape='rect' class='footnoteref' id='footnoteref-1' href='javascript:;' onclick='WIKIDOT.page.utils.scrollToReference(&apos;footnote-1&apos;)'>1</a>
</sup>一样物品或生物，且该物品对使用者来说具有情感价值<sup class='footnoteref'>
 <a shape='rect' class='footnoteref' id='footnoteref-2' href='javascript:;' onclick='WIKIDOT.page.utils.scrollToReference(&apos;footnote-2&apos;)'>2</a>
</sup>时，投入物会在触及项目底部时消失。在15秒~3分种后，SCP-CN-769内将会出现一个与投入物类型相同的物品。该物品将不再具有对于使用者的情感价值，但其功能性、质量、复杂程度等等会得到提升。目前认为提升所产生的额外货币价值等值于物品对使用者原有的情感价值，但SCP-CN-769是如何衡量二者的仍然未知。随着使用次数的增加，使用者会产生将更多物品投入项目的欲望。当前尚未确认这是出于使用者的贪欲还是项目异常效应的一部分。

项目被发现于浙江省██市██村，一场“旧物换新物”的集会引起了居住在该村的基金会外勤特工███的注意。在对该村村民进行了调查和访谈之后，外勤特工███认定一位村民口中“祖上遗留的宝物”为导致该现象的异常项目并向基金会进行了报告。随后前来的收容小组成功回收了项目并对该村村民进行了B级记忆删除。
