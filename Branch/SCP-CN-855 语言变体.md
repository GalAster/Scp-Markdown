# SCP-CN-855


**项目编号：** SCP-CN-855

**项目等级：** Euclid

**特殊收容措施：** 网络上的SCP-CN-855应被密切监视，机动特遣队MTF-丁亥-12 “粉刷匠” 将监视并清除网络存在的SCP-CN-855-2，收容疑似的SCP-CN-855-1，实物SCP-CN-855-2和SCP-CN-855-3。
SCP-CN-855-1应被收容于一个标准电子类异常物品收容柜内，对SCP-CN-855-1的测试只限D级人员进行，且不得超过60分钟。

所有SCP-CN-855-2必须被销毁，暴露于SCP-CN-855-2下的人类应进行标准B级记忆清除。

SCP-CN-855-3应处以D级记忆清除，记忆清除无效的应被收容于一个标准人型SCP收容间内，所有内部材料均应隔音处理，被收容的SCP-CN-855-3不得发放纸笔等书写工具。涉及SCP-CN-855-3收容和测试的工作人员在与其接触时应佩戴3级声音干扰装置。

FM 74.2电台已移交基金会管理，以██学院校园广播伪装。

**描述：**  SCP-CN-855是一类认知危害，表现为对语言/文字<sup class='footnoteref'>
 <a shape='rect' class='footnoteref' id='footnoteref-1' href='javascript:;' onclick='WIKIDOT.page.utils.scrollToReference(&apos;footnote-1&apos;)'>1</a>
</sup>中词语的随机替换，更换范围主要包括近义词，反义词<sup class='footnoteref'>
 <a shape='rect' class='footnoteref' id='footnoteref-2' href='javascript:;' onclick='WIKIDOT.page.utils.scrollToReference(&apos;footnote-2&apos;)'>2</a>
</sup>，字母/字符/文字的调换/局部变更<sup class='footnoteref'>
 <a shape='rect' class='footnoteref' id='footnoteref-3' href='javascript:;' onclick='WIKIDOT.page.utils.scrollToReference(&apos;footnote-3&apos;)'>3</a>
</sup>，偶尔会更换为毫无关联的词语或[数据删除]<sup class='footnoteref'>
 <a shape='rect' class='footnoteref' id='footnoteref-4' href='javascript:;' onclick='WIKIDOT.page.utils.scrollToReference(&apos;footnote-4&apos;)'>4</a>
</sup>，大段落的调换/变更/易位也会在该状态下发生。

SCP-CN-855-1是一类标准104键PC键盘，其品牌包括███，█████，██，███等11种。使用SCP-CN-855-1输入文字时，将导致SCP-CN-855的发生。经调查，SCP-CN-855的产生不早于SCP-CN-855-1的制造。

SCP-CN-855所输入的文档和打印件被定义为SCP-CN-855-2，长时间暴露于SCP-CN-855-2下将导致认识危害，暴露者将对受SCP-CN-855影响的文字产生认同，并在语言中产生SCP-CN-855。抄写/照打/复印SCP-CN-855-2的副本也存在SCP-CN-855。对SCP-CN-855-2进行模糊化处理或多次低质量复印可以减轻或消除异常效应，佩戴2型图像干扰眼镜也可以直接阅读SCP-CN-855-2而不感染SCP-CN-855。

语言中产生SCP-CN-855的1级异常的人类对象将视为SCP-CN-855-3，绝大多数SCP-CN-855-3无法顺利理解正常的语言词汇，因SCP-CN-855的1级和2级异常，部分语言词汇可以被SCP-CN-855-3理解。其书写的文字与SCP-CN-855-2具有相同异常效应。即使所使用的语言不同，SCP-CN-855-3之间也能寻找并确认同类。正常人类长时间暴露在SCP-CN-855-3的语言下将产生SCP-CN-855，以至转化为SCP-CN-855-3.广播/录音在一定保真度下也能传播SCP-CN-855，而在嘈杂环境或声音干扰下SCP-CN-855的传播将被限制甚至阻止。SCP-CN-855-3对使用正常语言的人类存在敌意<sup class='footnoteref'>
