# SCP-CN-293

**项目编号：** SCP-CN-293

**项目等级：** Safe

**特殊收容措施：** 一份1.9.11版本的SCP-CN-293拷贝被安装在一台iPad上，连接在标准测试间的充电基座上。标准测试间位于Site-CN-91，配备有12个多角度、高防护等级的摄像头；测试间带有落地玻璃窗，以便外部观察。标准测试间内的装修可以根据实验需求而调节；若得到3级人员的批准，也可以在其他不包含重要或者高危物品的房间进行测试。

目前，2级或以上的基金会员工可以在向项目组申请后，使用SCP-CN-293的1.9.11版本游戏不超过30分钟，使用过程应被记录并归档。若怀疑测试者在游戏过程中有过激行为，可以对以此为依据对其进行强制心理辅导。

为防止敏感信息泄露，SCP-CN-293的2.0及以上的版本则被加密，仅能在4级人员的监督下，令D级人员进行测试。

目前，基金会前台公司，惊喜制作收藏公司（Surprise Collection Production Inc. ）已利用好评漏洞等方式将App Store内的SCP-CN-293下架。www.████████.com的IP地址已被屏蔽以防止平民访问。该域名的注册者、软件的下载者等等仍在基金会的调查之中。若发现有平民下载了SCP-CN-293，将卸载后对其进行记忆删除处理。

**描述：** SCP-CN-293为一款LETTERS娱乐出品的“虐待小人Real”游戏，可以在安装了IOS或Android操作系统的掌上设备中使用。在被收容之前，该款游戏可以在App Store和一个地址为www.████████.com的网站中下载并安装，大小为131.2M。它的发布者身份未知，且仍然没有被追查到。

在启动后，屏幕上会出现LETTERS娱乐和“书虫”工作室的标志，之后出现游戏的主界面：一个简单的灰色背景，中间为醒目的彩色字显示游戏的名称“虐待小人Real”。如果启动游戏的人员（下称“用户”）没有独自处在一个封闭的房间中<sup class='footnoteref'>
 <a shape='rect' class='footnoteref' id='footnoteref-1' href='javascript:;' onclick='WIKIDOT.page.utils.scrollToReference(&apos;footnote-1&apos;)'>1</a>
</sup>，画面下方会有一行闪烁的黑色小字提示用户“请独处并关好门窗以开始游戏”。若用户独自处在一个封闭的房间中，画面下方则会显示“开始游戏”。

点击“开始游戏”后，一名约20岁、未辨认身份的男性人类个体，即SCP-CN-293-1，会从天花板出现并掉落在用户面前的地面；它不会因本次掉落受到伤害。与此同时，用户所在的房间会变得无法从外部进入，任何物理穿透的尝试均会遭遇到不明阻力。用户与正在运行SCP-CN-293的掌上设备（下称“设备”）会进入一种类似于“旁观者”的状态：他们无法受到房间内其他物品的影响，也可以与房间内其他人/物的位置产生重叠。房间内的所有物体都会被房间的边界阻挡。

屏幕上方会出现一系列风格极简的下拉菜单。用户可以通过选定其中一项交互功能，配合手部的动作，对SCP-CN-293-1实施暴力或者其他动作。以火烧为例，当被选择时，火焰将会从用户双手手掌的位置喷射而出。如果用户的手掌被阻挡，譬如手持、操作手机等设备，或者握拳，喷射便会暂停。其他交互功能也大多基于用户手部的动作和位置，它们不会伤害到用户本身。

SCP-CN-293-1会对受到的待遇做出相应的反应，一般为符合预期的普通人类的情绪反应，在使用部分“皮肤”时，也可能表现为夸张或戏剧化的反应。根据它的表现，SCP-CN-293能够意识到自己被另一个个体控制以用来取乐，但无法感知到用户的位置。它具有极高的再生能力，任何伤害均会在伤害源消失后快速愈合（速率可调节），并且始终不会因为外力被肢解。未观察到SCP-CN-293-1因伤死亡；若因特殊原因死亡，则它会在3秒后重新从天花板生成。

每当用户对SCP-CN-293-1施加暴力时，游戏便会给予用户一定的金钱奖励，取决于暴力造成的痛苦程度、戏剧性效果以及该种方式是否在先前被使用过。未知SCP-CN-293是如何量化或者判断上述内容的。用户可以使用金钱解锁新的交互功能。

在受到用户的交互动作的影响时，房间内的其它物品仍然表现得和现实生活中一致几乎一致<sup class='footnoteref'>
 <a shape='rect' class='footnoteref' id='footnoteref-2' href='javascript:;' onclick='WIKIDOT.page.utils.scrollToReference(&apos;footnote-2&apos;)'>2</a>
</sup>。然而，SCP-CN-293-1不会注意到或者尝试使用这些物品，它们似乎扮演了装饰物的角色。

此外，下拉菜单中可以更换“皮肤”。该功能可以使现有的SCP-CN-293-1从原处消失，一个新的SCP-CN-293-1个体从天花板掉落到地面，它的样貌和性格则取决于当前选择了何种“皮肤”。可被选择的生物个体往往与现实中存在的人或者知名的影视角色相符合；对现实中这些人的调查显示，他们与SCP-CN-293没有任何联系或者异常之处，也从未受到游戏过程的影响。用户可以使用金钱解锁新的皮肤。下拉菜单中也可以选择“设置”，以对游戏的各类参数进行设置。

屏幕中间会显示“上”“下”按钮，可调整用户的垂直高度；当受试者悬浮时，可以观察到他们似乎在一个看不见的平面上行走。屏幕也可能根据不同的交互功能而显示额外的按键，如“调整速度”等。

此外，游戏左下角有“复位房间”键和“结束游戏”键。当“复位房间”被点击时，房间内除了用户、运行SCP-CN-293的设备、SCP-CN-293-1以外的所有其他物品都会在3秒内恢复到游戏开始时的状态<sup class='footnoteref'>
 <a shape='rect' class='footnoteref' id='footnoteref-3' href='javascript:;' onclick='WIKIDOT.page.utils.scrollToReference(&apos;footnote-3&apos;)'>3</a>
</sup>。高速摄像机显示，这些物品的移动和生成轨迹似乎是它们在游戏开始或者上次复位之后，所有经历的快速“倒放”。SCP-CN-293-1可能会在过程中被飞掠的物体擦伤或者撞伤。

当“结束游戏”键被点击时，房间内所有物品将会恢复，SCP-CN-293-1消失，此后SCP-CN-293的一切异常现象停止，房间可以再次被进入。这也会发生在用户关闭运行中的SCP-CN-293程序、设备电量耗尽、设备意外毁坏时。若游戏正常退出，则不会留下任何长期的异常影响。
