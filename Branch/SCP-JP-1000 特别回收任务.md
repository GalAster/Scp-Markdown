**项目编号：** SCP-1000-JP

**项目等级：** [已删除]

**特殊收容措施：** 除非有特殊情况，否则禁止向探测器062-A发送信号。对小行星AL120进行轨道观测的观测站在通常情况下最少不能低于三个。如果观测到AL120的轨道发生变化，机动特遣队“暗淡蓝点”（PBD）应再次测算目标的杜林危险指数。如测算出目标的杜林危险指数高于七级，探测器062-A及其附属物应立即编入异常无效化的名册，所有对其的保护措施都将作废。有关此项目的资料和上述的F5表象区域<sup class='footnoteref'>
 <a shape='rect' class='footnoteref' id='footnoteref-1' href='javascript:;' onclick='WIKIDOT.page.utils.scrollToReference(&apos;footnote-1&apos;)'>1</a>
</sup>只限于授权人员阅读。

**描述：** SCP-1000-JP是为了采集小行星（2001-AL120）的样品为目的制造的深层太空探测器（标记编号为2003-062-A）的附属物。

目前绝不能对探测器062-A使用任何的远程操作。在AL120的表面再次发现探测器062-A以后，只要探测器062-A收到信号，对小行星AL120的轨道的异常影响现象就会发生。由基金会所属的16所观测站和221个民间观测站确认，轨道的变化确实和探测器062-A接收信号有关。观测到的变化是在探测器接收信号之后，小行星AL120发生轨道半径缩小、轨道速度减小等细微变化，但无论哪一个变化都会使发生天体碰撞的可能性提高。因为小行星AL120具有巨大的质量，有发生灾难性灭绝事件（ELE）的危险，所以目前禁止向探测器062-A发送信号。

2003年5月9日，为了执行采集小行星AL120样本的计划，发射了探测器062-A<sup class='footnoteref'>
 <a shape='rect' class='footnoteref' id='footnoteref-2' href='javascript:;' onclick='WIKIDOT.page.utils.scrollToReference(&apos;footnote-2&apos;)'>2</a>
</sup>基金会也承担了计划的费用，同时借出了一些器材供探测器使用。探测器发射并航行3.1AU<sup class='footnoteref'>
 <a shape='rect' class='footnoteref' id='footnoteref-3' href='javascript:;' onclick='WIKIDOT.page.utils.scrollToReference(&apos;footnote-3&apos;)'>3</a>
</sup>之后，遭遇到附近恒星爆发的耀斑（X28级），陷入了通信中断的状态。尽管基金会在随后进行了修复工作，但还是无法恢复与探测器的通信。最后在2006年12月31日，探测器062-A视为已损失，并停止其所有的工作。

失去探测690日之后，负责与探测器通信的研究所和基金会的CETI观测站（6EQUJ5）在频道8406.99MHz收到了微弱的信号。此信号在随后被判明是探测器062-A发送的信号，之后便确认发生了上述的异常现象，此项目已归为基金会管理。

作为替代计划选择的探测器（J102E）从再次发现062-A开始直到返回地球的数据已经记录，此替代探测器最终在进入地球大气圈时摩擦烧毁。
