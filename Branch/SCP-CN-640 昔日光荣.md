# SCP-CN-640


**项目编号** ：SCP-CN-640
**项目等级：** Keter

**特殊收容措施：** 项目应被六条不锈钢制成的金属链以吊挂方法收容在位于██省██市郊区的Site-CN-42的一间尺寸约为20M*20M*7M的收容隔间内，该收容间应处在站点的一独立建筑中，且必须与Site-CN-42主建筑大楼离开至少150米，必须建造多于五座备用收容隔间以防止收容突破。任何靠近附近的平民都必须进行C级记忆删除后释放。

当多于三名站点人员报告出现噩梦或多于十五名站点人员报告睡眠状况不佳或出现心悸时，项目应当立即被自动化无人机吊装并转移到另一备用收容隔间，运载项目的车辆应当为无人驾驶并且用多辆伪装车辆保护，站点特工必须确保没有任何智慧生命体接近运载中的项目三米以内。

每一座项目的收容建筑必须在外设立多层次拒马，铁丝网，哨卡，遥控武器站火力封锁并且必须有防空的备用收容隔间。Site-CN-42将借调一支炮兵部队进行对可能的敌军火炮射击进行火炮反制。收容建筑物内应有至少20名装备自动武器与反坦克武器的特工轮班看守，并二十四小时对建筑外进行监视，一般情况下不得观察收容中的项目，并且应当配备掺杂有SCP-148成分的认知保护头盔防止接近项目导致的认知危害。守卫收容隔间工作中任何表露出精神状况不佳的特工都必须撤换。无论如何，任何试图接近项目收容所的未被授权者都应当被就地处决。

任何关于项目的较详细信息均可按自己的安全等级询问4级站点安保主管Dr.████。安保主管在有任何职员资料申请时必须视其安全等级而定告知或部分告知询问者关于项目异常性质的详细信息以免其好奇心导致的接触试图甚至由此试图引发的收容失效。

**补充** ：在任何情况与任何防护下都禁止直视项目超过15秒，以防项目异常性质导致认知危害。

**描述** ：SCP-CN-640是一具███的尸体，推测为一黄种人，项目生物部分身高约在175cm左右（±1cm），其一臂与一腿均缺失，尽管距离项目被挖出已经经过了█年█月██日，但项目的生物部分依然没有被观察到出现腐烂迹象，任何对项目进一步的观察测量或描述均被证实会导致重度认知危害，因此，任何项目外形的详细复制品，包括照片，图画，对于外貌的细致说明文都不能被保留和阅读，此行为可能会导致严重模因感染而[资料删除]。项目的身体与一台不明机械设备相连，推测此为其异常性质的来源。

其异常性质将会在任何一名人员接近其3m的距离时被进一步触发，在此情况下项目将收容失效，当项目收容失效时将会造成[已编辑] 。

鉴于事故-640a，项目将永久停止任何实验。

**事故-640a：** 项目在Site-CN-42的E楼开挖地基时找到的一个地下防空洞中被发现。发现时项目仍处于惰性状态。当工程进行时，整个Site-CN-42的工作人员开始抱怨睡眠质量差劲，同时Site-CN-42附近的██省省会██市出现了“无法解释的睡眠质量下滑”和“呼唤向███运动的噩梦”并且多名市民开始自发向Site-CN-42运动，多次试图或确实到达E楼工地附近。由于当时的天气状况不佳，基金会在施工中使用了自动挖掘机，因此没有导致项目的异常性质被进一步触发，特工得以通过自动化手段收容项目。在收容中有3位特工受到了程度不一的认知危害。试图存放项目的准备过程中，研究员助理██████饲养的一只家猫自行闯入了项目的临时存放地点，并过度接近项目导致了项目的异常效应被部分触发，项目[资料删除]，将部分在场工作人员连同家猫[已编辑]，并有多达██名基金会工作人员受到严重的认知危害，成为重度精神病人或植物人，███名基金会人员受到等级不同的认知危害。基于分析猫的智力与项目的异常特性导致的后果，推测若多于三名人类进入于项目的异常触发范围内，后果可能导致一次AK级疯狂末日，鉴于项目通过噩梦和微量精神影响来在潜意识中广播自身以此吸引一定范围内有一定智慧的生物接近到项目附近并与之接触的效应，不排除项目被任何GOI发现并组织袭击的可能性，故安排了现有的多重收容措施。
