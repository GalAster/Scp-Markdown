# SCP-CN-284

**项目编号** ：SCP-CN-284

**项目等级** ：Euclid

**特殊收容措施** ：SCP-CN-284的收容间分为两部分，第一部分为一间200立方米的混凝土隔间，第二部分为一间10立方米的注满水的顶部不透明玻璃隔间，玻璃隔间外应加装有足够的防撞装置，并且使用数个磁悬浮装置使玻璃隔间悬浮在混凝土隔间正中央。

每天为SCP-CN-284喂食一块1.2kg以上的肉类，喂食时将顶部的夹层打开放入肉类，当安放人员离开后遥控打开隔间，并且确保SCP-CN-284食用完全。

对于SCP-CN-284进食时产生的“震动”，确保其控制在可控范围内。

**描述：** SCP-CN-284外观为一艘长4米，宽1米的木制船只，经推测可能为【数据删除】时期的造物，SCP-CN-284在接近船头的位置出现了锯齿状可以进行咬合动作的嘴部，而船尾和船身部分也分别出现了类似鲨鱼鱼尾以及鱼鳍的木质结构。

SCP-CN-284可以进行游动，潜水等类似正常鱼类的动作，但SCP-CN-284似乎更偏向于以正常船只的方式进行活动。

SCP-CN-284需要每天投放1.2kg以上的任意肉类，根据已知数据，如果超过74小时没有进食，SCP-CN-284会反复撞击收容间并且企图逃脱收容直到重新进食为止，目前不建议对SCP-CN-284进行进一步的饥饿试验以防【数据删除】。

当SCP-CN-284发现“猎物”时，SCP-CN-284会将猎物“锁定”，在此时无论被锁定物的速度如何，SCP-CN-284永远保持了比该速度高20cm/s的速度前进直到捕食完成。
